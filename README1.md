# MENTAL HEALTH IN TECH INDUSTRY
## Objectives:

1. Load the Data from the dataset
2. Understand Data Connections between the tables
3. Identify Sociodemographic Features of the dataset
4. Identify Potential Bias
5. Answer Key Questions:
   - Does Company Size Impact Mental Well-Being?
   - Does the Tech Industry Exhibit More Mental Health Issues Compared to Other Industries?
   - Do Workplace Mental Health Discussions Encourage Employees to Share Issues with Supervisors?
   - What Are the Most Common Mental Health Conditions?
6. Provide Recommendations

## Methods Used:
* SQL Queries: Extracted and joined data from multiple tables, performed calculations and aggregations.
* Data Cleaning: Used Pandas to handle missing values, convert data types, and rename columns.
* Analysis: Conducted initial analysis in SQL and further analysis with Pandas and Numpy.
* Visualization: Created bar plots and stacked bar charts using Matplotlib and Seaborn.
* Documentation: Documented the process for clarity and reproducibility.
